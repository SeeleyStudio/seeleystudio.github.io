Animatext

Animatext is a simple, useful and powerful text animation system for Unity. You can quickly and efficiently make text animations including titles, dialogs, introductions, numbers, cutscenes and so on.

Full Version Free Version

Rich text tags

Just like using rich text, you can add specified tags to the text of the text component to realize the corresponding text effect in the animation system.

Flexible character combinations

The animation system includes not only character, word, line mode, but also customized group mode. In any mode, you can use tags to redefine unit combinations of the characters.

Customizable presets

Every preset in the animation system contains rich and user-friendly parameters. You can easily get amazing text presets with simple modifications in just a few minutes.

Controllable effects

The animation system contains common effect control functions. You can use codes to control the state, time and speed of the effect etc.
